# 🌍 Mundial Económico

O Mundial Económico é uma aplicação web full-stack desenvolvida como um marketplace multi-vendedor, onde diferentes utilizadores podem gerir os seus próprios produtos de forma independente. A plataforma foi construída com foco em escalabilidade, organização de código e experiência do utilizador.

---

## 🚀 Funcionalidades

- 🔐 Autenticação de utilizadores e gestão de sessões  
- 🛍️ Gestão de produtos por múltiplos vendedores  
- ➕ Criação, edição e remoção de produtos (CRUD)  
- 📦 Arquitectura escalável para múltiplas lojas  
- 🌐 Interface moderna e responsiva

## 🏗️ Arquitectura

A aplicação segue uma arquitectura modular:

- **Frontend:** Responsável pela interface e interacção com o utilizador  
- **Backend:** Responsável pela lógica de negócio, autenticação e processamento de dados  
- **Base de Dados:** Armazena utilizadores, produtos e dados dos vendedores  

Cada vendedor opera de forma independente, garantindo separação de dados e maior escalabilidade.
